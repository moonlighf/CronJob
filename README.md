## CronJob

### [1.epic_game](https://github.com/moonlighf/CronJob/tree/main/epic_game)
&#8195;获取Epic的每周免费游戏资讯，并实现游戏的自动领取入库 **（默认关闭，如果开启推荐用小号，防止垃圾游戏污染游戏库）**
